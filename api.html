¿Qué es una llamada a la IA?

Cada vez que un programa le manda información a Claude y recibe una respuesta, eso es una llamada. Es como una consulta: tú preguntas, la IA responde, y eso tiene un costo de tokens.

Normalmente necesitas una API Key propia para hacer esto. Con este servicio, usas la mía — tú solo pagas por lo que consumes, sin configuraciones técnicas.

Preguntas frecuentes
¿En qué programas puedo usar las llamadas? +
En todos los programas que construiste en la clase magistral de AgentIA, y en cualquier programa generado con nuestro generador de prompts.
¿Qué pasa si subo un archivo grande con muchas filas? +
Si el texto supera 2,000 palabras se descuentan 2 créditos en lugar de 1. El programa te avisa antes de procesar para que decidas.
¿Los créditos tienen fecha de vencimiento? +
No. Los créditos de pago único (Starter y Pro) no vencen. Solo el plan Ilimitado se renueva mensualmente.
¿Mi información es segura? +
Sí. Tu API Key nunca se expone — todo pasa por nuestro servidor seguro. El texto que procesas va directo a Claude y no lo almacenamos.
¿Hay factura (CFDI)? +
Sí. Envía tus datos fiscales a hola@argentia.agency después de tu compra.
